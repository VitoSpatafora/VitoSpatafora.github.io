CSCE 242 Web Applications

Vito Spatafora

This course introduces the fundamentals of web development and client-server computing. Students will learn to create and style dynamic websites using HTML, CSS, and JavaScript, while also developing object-oriented programming skills in a JavaScript environment. The course covers data parsing with JSON, as well as server-side development using Node.js, Express, and MongoDB. Emphasis is placed on integrating client-side and server-side applications to exchange data effectively. By the end of the course, students will have the skills to design, develop, and deploy interactive, data-driven web applications.
